# 📝 HTML & CSS List Website

## 📖 Description

This project is a simple website created using **HTML** and **CSS**. It demonstrates how to create and style different types of lists while improving the user interface with a moving header animation. The project is designed for beginners who want to learn the basics of web development and CSS animations.

## ✨ Features

* Create ordered and unordered lists using HTML.
* Style lists with CSS.
* Animated moving header for an attractive look.
* Clean and responsive layout.
* Beginner-friendly project structure.

## 🛠️ Technologies Used

* HTML5
* CSS3

## 🚀 How to Run

1. Download or clone this repository.
2. Open the project folder.
3. Double-click the `index.html` file or open it in your favorite web browser.

## 📂 Project Purpose

This project helps beginners understand how to:

* Build web pages using HTML.
* Style elements with CSS.
* Create attractive animations using CSS.
* Improve the appearance of a webpage with simple design techniques.
